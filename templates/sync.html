📥 Install all sync bookmarks at once

Instead of dragging each button below to your bookmarks bar one at a time, download them all in one folder. Re-download anytime to update them to the latest version.

  1. Click the button — it saves acculynx-crm-bookmarks.html.
  2. In Chrome open ⋮ → Bookmarks & lists → Bookmark manager (or chrome://bookmarks).
  3. Click ⋮ (top right) → Import bookmarks, pick that file. A folder “AccuLynx → CRM Sync” appears with all 10 tools.
  4. To update later: re-download here and import again (drag the old folder to Trash first if you want a clean swap).
